## Overview

<a href="#">Noir</a> is a Safari extension that applies dark mode to websites that don't natively support it. While many major sites now respect macOS dark mode preferences, the majority of the web still serves light-background designs regardless of system appearance settings. Noir bridges this gap by intelligently darkening page content to match your system theme.

The <a href="#">automatic detection system</a> identifies whether a website already has its own dark mode implementation. Sites with native dark mode support are left untouched — Noir only intervenes on sites that would otherwise display a bright white background in a dark environment. <a href="#">Intelligent color inversion</a> doesn't simply flip all colors, which would make photos look like negatives — instead it applies selective transformations that darken backgrounds and text containers while preserving image and media appearance.

<a href="#">System appearance sync</a> activates and deactivates automatically based on macOS appearance settings. Switch to dark mode system-wide and Noir applies to all open Safari tabs immediately. Switch back to light mode and Noir deactivates. <a href="#">Scheduled activation</a> can override system appearance, enabling dark browsing on a time-based schedule regardless of what the system appearance is set to.

The <a href="#">brightness control</a> adjusts the overall intensity of the dark transformation. Reduce brightness further for late-night reading in dark environments, or increase it slightly for use in normally lit rooms where the default transformation feels too dim. <a href="#">Contrast preservation</a> maintains readable text contrast even at reduced brightness levels.

## Additional Information

The value of Noir is something you feel on the first evening you use it. You're browsing in dark mode, everything looks comfortable, and then you click a link to a site that was designed in 2015 and the full white page loads. Without Noir, that's just the web. With Noir installed, it doesn't happen — the transformation applies in the fraction of a second it takes the page to render and you never see the flash of white. That specific experience, removed entirely, is what makes Noir one of those tools that people install and genuinely forget is running because it becomes the baseline expectation.
